 .. _filebrowser:

File Browser
======================================================================

This is a multi-site extension of `Django/Grappelli File Browser 
<https://github.com/sehmaschine/django-filebrowser>`_.
Main features:

- new base class for file sites
- site identified by subdirectory name
- custom file extensions
- permission to access fielbrowser views
